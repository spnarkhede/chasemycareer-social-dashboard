## 🎯 Overview

**Chase My Career Social Media Manager** is an enterprise-grade, AI-powered social media management platform designed specifically for career professionals, coaches, and businesses in the career development niche.

Built with modern technologies and best practices, it provides everything needed to manage, automate, and optimize social media presence across 9+ platforms from a single dashboard.

### 🎯 Key Value Propositions

- **Save 20+ hours/week** with AI-powered content creation and automation
- **Increase engagement by 300%** with optimized posting times and AI suggestions
- **Manage all platforms** from one unified dashboard
- **Monetize your audience** with built-in monetization strategies
- **Track competitors** and stay ahead of industry trends
- **Enterprise-ready** with team collaboration and advanced security

## 🚀 Getting Started

### Prerequisites

```bash
- Node.js 18+ (https://nodejs.org/)
- npm or yarn
- PostgreSQL 15+ (or Supabase/Neon)
- Git
```

### Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/chasemycareer-social-dashboard.git
cd chasemycareer-social-dashboard

# 2. Install dependencies
npm install

# 3. Copy environment file
cp .env.example .env.local

# 4. Generate security secrets
openssl rand -base64 32  # NEXTAUTH_SECRET
openssl rand -hex 32     # ENCRYPTION_KEY
openssl rand -hex 16     # CRON_SECRET

# 5. Update .env.local with your credentials
# (See Configuration section below)

# 6. Setup database
npx prisma generate
npx prisma migrate dev
npx prisma db seed

# 7. Start development server
npm run dev

# 8. Open browser
# http://localhost:3000
```

### Using Setup Script

```bash
# Windows PowerShell
.\setup.ps1

# Linux/Mac
chmod +x setup.sh
./setup.sh
```

### Base URL

```

```

### Authentication

All API requests require authentication:

```bash
# JWT Token
Authorization: Bearer <jwt_token>

# API Key
X-API-Key: cmc_<your_api_key>
```

### Key Endpoints

#### Authentication
```http
POST   /api/auth/signin          # Sign in
POST   /api/auth/signup          # Sign up
POST   /api/auth/signout         # Sign out
POST   /api/auth/forgot-password # Request password reset
POST   /api/auth/reset-password  # Reset password
POST   /api/auth/2fa/setup       # Setup 2FA
POST   /api/auth/2fa/verify      # Verify 2FA
```

#### Posts
```http
GET    /api/posts                # Get all posts
POST   /api/posts                # Create post
GET    /api/posts/:id            # Get post
PUT    /api/posts/:id            # Update post
DELETE /api/posts/:id            # Delete post
POST   /api/posts/bulk           # Bulk actions
POST   /api/posts/:id/publish    # Publish post
```

#### Analytics
```http
GET    /api/analytics            # Get analytics

GET    /api/analytics/sentiment  # Sentiment analysis
GET    /api/analytics/trends     # Trend data
GET    /api/analytics/roi        # ROI tracking
```

#### Automation
```http
GET    /api/automation/rules     # Get automation rules

PUT    /api/automation/rules/:id # Update rule
DELETE /api/automation/rules/:id # Delete rule
POST   /api/automation/queue/process  # Process queue
```

#### AI
```http
POST   /api/ai/generate          # Generate content
```
